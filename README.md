# Gyudon

> A online space for my shenanigans, taken from [jez's](https://www.github.com/jez) [tufte-pandoc-jekyll theme](https://github.com/jez/tufte-pandoc-jekyll).

## Setup

```
$ brew install pandoc
$ brew install pandoc-sidenote
$ mkvirtualenv thoughts
$ gem install bundler
$ bundle install
```

## Using

```
octopress new draft

octopress publish

octopress new page
```
