# Gyudon

A place for my thoughts and write-ups. Uses [jez](https://www.github.com/jez)'s [Tufte CSS theme](https://github.com/jez/tufte-pandoc-css).

## Setup

```
$ brew install pandoc
$ brew install pandoc-sidenote
$ mkvirtualenv thoughts
$ gem install bundler
$ bundle install
```

## Using

```
octopress new draft

octopress publish

octopress new page
```
